# Blazer


This is the server of my web app, a social media that brings travelers and travel companies together.
I used **ReactJS** and **NodeJS** to build my App.

Database **MongoDB**

## tools
- express
- jsonwebtoken
- mongoose
- multer
- uuid
- bcryptjs
- dotenv
- cors
